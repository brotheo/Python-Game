Question 1
What is the primary purpose of the tkinter module in Python?
A) Creating web applications
B) Creating graphical user interfaces (GUIs)
C) Working with databases
D) Processing images
Correct! Tkinter (Tk interface) is Python's standard GUI toolkit for creating desktop applications with windows, buttons, labels, and other graphical elements.
Question 2
Which method is essential for keeping a Tkinter window open and responsive?
A) window.start()
B) window.run()
C) window.mainloop()
D) window.display()
Correct! The mainloop() method starts the event loop that keeps the window open and responsive to user interactions. Without it, the window would appear and immediately close.
What does this code create?
import tkinter as tk window = tk.Tk() window.title("My App") label = tk.Label(window, text="Hello World!") label.pack() window.mainloop()
A) A window with a button
B) A window with a text input field
C) A window with a "Hello World!" label
D) An empty window
Correct! This code creates a window titled "My App" with a Label widget displaying "Hello World!". The pack() method adds the label to the window.
Question 4
Which widget is used to get text input from the user?
A) Label
B) Button
C) Entry
D) Frame
Correct! The Entry widget creates a text input field where users can type text. You can retrieve the text using the .get() method.
Question 5
What happens when you click a Button with command=my_function?
A) The button changes color
B) The function my_function is called
C) The window closes
D) Nothing happens
Correct! The command parameter specifies which function to call when the button is clicked. This is how you handle button click events in Tkinter.
Question 6
Which Canvas method is used to draw a circle?
A) create_circle()
B) create_oval()
C) create_round()
D) draw_circle()
Correct! The create_oval() method draws an oval shape. When the width and height are equal, it creates a perfect circle.
Question 7
What does entry.get() return?
A) The Entry widget object
B) The text typed in the Entry widget
C) The position of the cursor
D) True or False
Correct! The .get() method retrieves the text that the user has typed into the Entry widget as a string.
What's wrong with this code?
import tkinter as tk window = tk.Tk() button = tk.Button(window, text="Click me!") button.pack() # Missing something here...
A) Missing window.title()
B) Missing window.mainloop()
C) Missing command parameter
D) Nothing is wrong
Correct! The code is missing window.mainloop() at the end. Without it, the window would appear briefly and then close immediately.
Question 9
In the calculator project, what does StringVar() help with?
A) Converting strings to numbers
B) Automatically updating label text when the variable changes
C) Creating string objects
D) Validating user input
Correct! StringVar() creates a special variable that automatically updates any widgets connected to it when its value changes, making it perfect for displaying results.
Question 10
What makes the click game challenging and fun?
A) Random button positioning and time pressure
B) Complex graphics
C) Multiple windows
D) Database integration
Correct! The click game uses random positioning to move the button around the screen and includes a timer to create time pressure, making it both challenging and engaging!
